Hex Code란?<br/>
'0-9까지의 숫자'와 'A~F'까지의 알파벳이 랜덤하게 구성되어 이루는 6자리 코드를 의미합니다.<br/>
예를 들면 000000, 3474FF 등 모두 유효한 Hex Code입니다.<br/> 
CSS에서는 Hex Code앞에 #를 붙여 색상값으로 이용할 수 있습니다.<br/>
예) #3474FF<br/>
